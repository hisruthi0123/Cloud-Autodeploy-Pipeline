# Cloud AutoDeploy Pipeline

## Overview
This project focuses on building an automated CI/CD pipeline to deploy applications reliably to cloud infrastructure. It was created to gain hands-on experience with DevOps workflows, automation, and deployment troubleshooting.

## Tech Stack
- Cloud: AWS EC2
- Containers: Docker
- CI/CD: GitHub Actions
- Web Server: Nginx

## Core Features
- Automated build and deployment pipeline
- Containerized application deployment
- Cloud-based hosting and configuration
- Basic monitoring using logs

## Engineering Focus
- Automating deployments using CI/CD pipelines
- Debugging deployment failures using logs
- Managing containerized services
- Documenting deployment workflows and fixes
